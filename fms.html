FileMaker Server에 관하여

FileMaker ® Server는 FileMaker Pro Advanced 파일 을 열고 네트워크의 클라이언트가 사용할 수 있도록하고 웹 페이지 나 다른 응용 프로그램에 FileMaker 데이터를 발행 하는 전용 데이터베이스 서버입니다 .

FileMaker Server 관리 콘솔

FileMaker Server 관리 콘솔은 FileMaker Server를 구성하고 관리 할 수있는 웹 응용 프로그램입니다. 호스팅 된 데이터베이스에서 데이터를 처리, 모니터링 및 공유합니다. 연결된 클라이언트를 관리합니다.

FileMaker Server가 실행중인 동일한 컴퓨터 또는 FileMaker Server에 네트워크로 액세스 할 수있는 거의 모든 Windows 또는 macOS 컴퓨터에서 관리 콘솔을 실행할 수 있습니다. 관리 콘솔 시작을 참조하십시오 .

지원되는 고객

  • FileMaker Pro 16 Advanced, FileMaker Pro 16, FileMaker Pro17 Advanced 및 FileMaker Pro 18 Advanced
  • FileMaker Go 16, FileMaker Go 17 및 FileMaker Go 18
  • 파일 메이커 서버를 통해 데이터에 액세스하는 웹 사용자 웹 게시 엔진을 사용하여 일 메이커 WebDirect , 사용자 정의 웹 게시 XML과 PHP와, 및 사용자 정의 웹 게시를
  • 호스팅 솔루션의 데이터에 액세스하기위해 FileMaker Data API 를사용하는 웹 비스 또는 응용 프로그램
  • FileMaker 클라이언트 드라이버를 사용하는 ODBC (Open DatabaseConnectivity) 및 JDBC (Java Database Connectivity) 클라이언트
